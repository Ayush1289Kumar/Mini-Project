## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Ayush1289Kumar/Mini-Project.git
```

### Navigate to the project

```bash
cd Mini-Project
```

### Run Java Projects

Compile:

```bash
javac FileName.java
```

Run:

```bash
java FileName
```

### Run Python Projects

```bash
python filename.py
```

or

```bash
python3 filename.py
```
